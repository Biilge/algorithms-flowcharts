# Task 2 – Online Shopping Cart System

## System Description

This task models an online shopping system including product browsing, cart management, and checkout operations.

## Functional Requirements

The system includes the following steps:

- User login verification
- Browsing product categories (loop)
- Adding products to the cart
- Stock availability validation
- Viewing and editing the shopping cart
- Applying discount codes
- Minimum order amount validation ($10)
- Shipping fee calculation (free shipping above $40)
- Payment method selection
- Order confirmation

## Algorithm Design

The algorithm was designed using structured pseudocode including:

- START and END statements
- READ and PRINT operations
- IF–THEN decision structures
- LOOP constructs for browsing and cart management

## Flowcharts

Two diagram formats were created:

- Graphviz DOT
- Mermaid Flowchart

Both diagrams follow standard flowchart conventions for processes, decisions, and input/output operations.
